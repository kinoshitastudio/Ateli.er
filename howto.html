— 01 / MY CHANNEL

My Channel — 自分の作品置き場 My Channel — your own shelf

自分が作った channel と、その中に置いた block の一覧。アトリエの中で「自分の場所」になります。

A list of channels you've created and the blocks you've placed inside them. Your space within Atelier.

わかりづらいポイント:
「My Channel」の名前にハンドル名 (例: Taka) が出るので、自分が誰として見えてるかがわかります。タイトル直下に 6 channels · 23 blocks のように数字が出ます。グリッドはすべて自分が置いた block。 Tip:
Your handle (e.g. Taka) is shown in the title, so you know how others see you. Right under the title: 6 channels · 23 blocks. Every card in the grid is something you placed.
channel channel = 連作のテーマ。ひとつの作品郡を入れる "枠"。 channel = a theme. A "shelf" for a related set of works.
block block = 1 つの作品。audio / image / text / url の 4 種類のいずれか。 block = a single work. One of: audio / image / text / url.
edit — edit でグリッドの並び替え、 channel 名の変更、削除など。 — edit to reorder, rename channels, delete blocks, etc.
— 02 / FEED

Feed — 全 channel の活動を時系列で Feed — all activity, newest first

自分のすべての channel に置いた block を 新しい順 にまとめたタイムライン。アルゴリズムなし、ランキングなし、いいね数なし。flat な記録 です。

A flat, chronological timeline of every block you've placed — newest first. No algorithm. No ranking. No like counts. Just a record.

例: @Taka placed 2 images in BMboard のように、誰が・どの channel に・何を置いたかが 1 行で出ます。

Format: @Taka placed 2 images in BMboard — who, what, where, in one line.

— 03 / EXPLORE

Explore — 他の住人を見にいく Explore — visit other tenants

アトリエに住む 他の人たち (Residents) と、彼らが Public に公開した block をまとめて見る場所。

A list of other residents and the blocks they've made public.

わかりづらいポイント:
「My Channel」が 自分 の作品なら、「Explore」は 他人 の作品。residents セクションでハンドル名と参加日が、その下に public blocks の thumbnail が並びます。クリックで block 詳細へ。 Tip:
My Channel = your own. Explore = everyone else. The residents row shows other users' handles + join date; below is a grid of their public blocks. Click any to see the full block.
residents アトリエに住む人のリスト。ハンドル + 参加日。クリックでその人の public channel へ。 List of residents — handle + join date. Click to see their public channels.
public 他の人が Public に設定した block のみ がここに出ます。Draft はオーナーだけが見られる。 Only blocks set to Public appear here. Drafts stay with the owner.
silent フォロー機能なし、いいねなし。静かに見にいくだけの場所。 No follow, no like. Just a place to quietly look in.
— 04 / COMMENT

Comment — 短いひとこと Comment — single-line replies

自分の block に届いた 短いコメント の一覧。アトリエでは「コメント = ひとこと」と決めています。 返信通知も、ネストも、長文もありません。短く・ひとつだけ。

A list of short single-line replies placed on your blocks. No reply threads. No nesting. No long-form. Just brief, singular.

画像のように @DDD「後で聞きます！」 といった一言が時系列で並びます。

As shown: @DDD "後で聞きます!" — listed in time order.

— 05 / UPLOAD

Upload — block を置く Upload — place a block

アトリエの central な動作: block を channel に置く こと。 Atelier は完成品を売る場ではなく、何かが生まれる現場です。 ここはあなたの制作物を、静かに置いておく場所です。

The central action: place a block in a channel. Atelier isn't a place to sell finished work — it's a working site. Leave things here, quietly.

わかりづらいポイント (1):
あなたの channel と block は Supabase 上に保存されます (アカウント連携)。
同時にこのブラウザにも localStorage でローカルキャッシュとして残っているので、オフラインでも自分のものは見られます。
もし別端末に ローカルキャッシュごと 移したい場合は、画面上部の SYNC TO ANOTHER DEVICE から URL を発行 → 移行先で貼り付けてください (補助機能)。 Tip 1:
Your channels and blocks are saved on Supabase (linked to your account).
They're also cached in this browser's localStorage so you can browse your own offline.
To migrate the local cache across devices, use SYNC TO ANOTHER DEVICE at the top to generate a URL and paste it elsewhere (an optional helper).

画面を下にスクロールすると 「add a block to a channel」 フォームが出ます。

Scroll down to find the "add a block to a channel" form.

わかりづらいポイント (2):
上部に AUDIO LINK · IMAGE LINK · TEXT · URL の 4 タブ。これが block の 4 種類 です。先に種類を選ぶ → channel を選ぶ → タイトル等を入力 → 保存。

AUDIO は Google Drive / Dropbox / 直接 mp3 url のような 外部リンク方式。アップロードではなく URL を貼ります。 Tip 2:
4 tabs at the top: AUDIO LINK · IMAGE LINK · TEXT · URL. These are the 4 block types. Choose a type → pick a channel → fill in the title and details → save.

AUDIO uses external links (Google Drive / Dropbox / direct mp3 url). You paste a URL, not upload a file.
type type = block の種類。AUDIO LINK / IMAGE LINK / TEXT / URL。 type = block kind. AUDIO LINK / IMAGE LINK / TEXT / URL.
channel channel = どの "枠" に入れるか。既存から選ぶ or + new で新しく作る。 channel = which "shelf" to put it in. Pick an existing one or + new.
link audio url / image url は 外部のリンク。Google Drive / Dropbox の直リンクを貼ってください。 audio url / image url are external links. Use direct links from Google Drive / Dropbox.
— 06 / CHANNEL VIEW

Channel view — 連作の中 Channel view — inside a series

My Channel から channel を 1 つ選ぶと、その中の block 一覧 が出ます。 画像の例は Taka / Demos / 99letters demos — 6 blocks。 パンくず (Taka / Demos) で 1 つ上の階層に戻れます。

Tap a channel from My Channel to see the blocks inside it. In the example: Taka / Demos / 99letters demos with 6 blocks. Use the breadcrumb (Taka / Demos) to step back.

block の右上の □ アイコンは Public / Draft の表示。Public は他の住人にも見える、Draft は自分だけ。

The □ badge on each block shows Public / Draft state. Public is visible to residents; Draft is only you.

— 07 / BLOCK VIEW

Block view — 1 つの作品ページ Block view — a single work

block 1 つを開いた画面。パンくず + メタ情報 (channel, blocks 数) + 本体 (画像 / 音 / テキスト / URL) + コメント欄。 AUDIO block の場合は PLAY HERE ボタンで再生開始。再生中は画面下にプレイヤーバーが固定されます。

A single block opened. Breadcrumb + meta (channel, block count) + body (image / audio / text / url) + comments. For AUDIO: tap PLAY HERE. A player bar fixes to the bottom while playing.

— 08 / CONNECT (core feature)

Connect — 他人の作品を自分の channel に呼ぶ Connect — pin another's work into your channel

Atelier ならではの機能 「connect」。 Explore で見つけた 他人の public block を、自分の channel に 紐付ける (pin) 動作です。 reblog / repost と似ているけど、並べる側の文脈を主役にする違いがあります。

The core feature: connect. Pin another resident's public block into your own channel. Similar to reblog / repost — but the emphasis here is on your context of placement, not the reblog itself.

たとえば 「@DDD の音源」と「@BMBoard の画像」を、 自分の channel "影響源" に並べる。すると、複数の住人の作品が、あなたの編集行為によって 新しい意味を持つ。これが connect の本質です。

Example: pin @DDD's audio and @BMBoard's image into your channel called "influences". The blocks gain new meaning through your editorial act — this is the core of connect.

connect の使い方:
1. Explore で気になる public block を開く
2. 画面の → connect ボタンをタップ
3. 自分のどの channel に紐付けるかを選ぶ
4. 自分の My Channel に "pin した block" として現れる

元の block は所有者のもののまま。あなたは 選んで並べただけ。それがアトリエの "編集者性" です。 How to use connect:
1. Open a public block in Explore
2. Tap the → connect button
3. Pick which of your channels to pin it into
4. It appears in your My Channel as a "pinned block"

The original block stays with its owner. You chose and placed it — that's the "editorial act" Atelier is built around.
