## About this service

Not a place to sell finished work, nor a torrent like social media.
A place to **leave things, still unfinished**.
No algorithms. No rankings. No like counts.

**Keywords**: `courtyard` · `unreleased` · `human signature` · `no algorithm` · `interconnected depth` · `invitation-based`

For the philosophy and concept, see [about page](https://atelistudio.com/about.html).

## Four units

1. **block** — the unit. One of: audio / image / text / url
2. **channel** — a container for blocks. A series, a theme, a thought in progress
3. **connect** — pin another user's public block into your channel (the editorial act)
4. **footprint** — a brief single-line comment on a block. No reply threads, no nesting

---

## Main features

### Block & channel management
- Add / edit / delete blocks (individual + bulk)
- Create / edit channels; switch state (current / future / past)
- Reorder blocks within a channel (edit mode)
- Search within channel / generate share link

### Navigation
- Home: 3 marquees (catchphrase / title / image)
- Block detail modal (a personal atelier)
- Inline audio player
- Per-block comments (a "trace" that fades over time)
- Feed: chronological stream of all blocks
- 5 tabs: My Channel / Feed / Explore / Comment / Upload

### Cross-device
- Same channels and blocks accessible from any device with your account
- Changes reflect immediately, saved to Supabase in the background

### Public / Explore
- Mark a block **Public** to surface it in Explore
- Browse other residents' public blocks; pin interesting ones via **Connect**
- A connected block is a snapshot — even if the original author edits or deletes, your copy remains

### Cross-user comments
- Leave a single-line comment on another user's block
- A red dot in the top-right notifies you when someone comments on yours
